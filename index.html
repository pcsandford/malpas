What is the pattern arcade?

The Pattern Arcade is a collection of backgrounds that you can use for your own website or print projects. Each pattern can be downloaded as either a .png or .svg file. The .png can be used in your CSS to tile to fill any background size, while the .svg can be modified to match your own colour scheme.
