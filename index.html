What is the pattern arcade?

The Pattern Arcade is a collection of backgrounds that you can use for your own website or print projects. Each pattern can be downloaded as a PNG file. The PNG can be used in your CSS to tile to fill any background size.

A future update will include the option to download SVG files so you can add custom colours and create your own patterns.
